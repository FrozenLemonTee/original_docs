Detailed Description

template<typename TYPE, template< typename > typename SERIAL = chain>
class original::queue< TYPE, SERIAL >

First-In-First-Out (FIFO) container adapter <TYPE, SERIAL>

Template Parameters
TYPEType of elements stored in the queue
SERIALUnderlying container type (default: chain), see constraints from original::containerAdapter

Implements queue operations using the specified underlying container. Supports insertion at back and removal from front. Inherits template constraints from original::containerAdapter.
