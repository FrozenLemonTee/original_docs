Detailed Description

template<typename TYPE, template< typename > typename SERIAL = chain>
class original::deque< TYPE, SERIAL >

Double-ended queue container adapter <TYPE, SERIAL>

Template Parameters
TYPEType of elements stored in the deque
SERIALUnderlying container type (default: chain), see constraints from original::containerAdapter

Implements deque operations using the specified underlying container. Supports efficient insertion and removal at both front and back ends. Inherits template constraints from original::containerAdapter.
