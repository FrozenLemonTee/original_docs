Detailed Description

Base class for reference-counted smart pointers.

Implements core functionality for automatic memory management using reference counting. Supports strong/weak references and custom deleters. Inherits printable and comparable interfaces for debugging and comparison.

Key Features:

  • Automatic memory management via reference counting
  • Support for both strong and weak references
  • Customizable deletion policies via template parameter
  • Thread-safe reference counting operations
  • Integration with printable, comparable and hashable interfaces
Note
This is a base class meant to be inherited by concrete smart pointer implementations
