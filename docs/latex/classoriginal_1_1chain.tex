\subsubsection*{template$<$typename TYPE$>$\newline
class original\+::chain$<$ TYPE $>$}
Non-\/cyclic doubly linked list container. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Type of elements stored in the list\\
\hline
\end{DoxyTemplParams}
Implements a classic doubly linked list with\+:
\begin{DoxyItemize}
\item Sentinel nodes for boundary management
\item Bidirectional traversal capabilities
\item Index-\/based element access (O(n) complexity)
\item Deep copy semantics 
\end{DoxyItemize}
