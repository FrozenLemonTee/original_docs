\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE$>$\newline
class original\+::random\+Access\+Iterator$<$ TYPE $>$}
Abstract base class for random-\/access iterators. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Type of elements being iterated\\
\hline
\end{DoxyTemplParams}
Implements core functionality for iterators that support\+:
\begin{DoxyItemize}
\item Pointer arithmetic operations (+, +=, -\/, -\/=)
\item Bidirectional traversal
\item Position validity checking
\item Element access/modification
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Maintains three state markers\+:
\begin{DoxyItemize}
\item Raw pointer to the current element
\item Reference to parent container
\item Absolute position index 
\end{DoxyItemize}
\end{DoxyNote}
