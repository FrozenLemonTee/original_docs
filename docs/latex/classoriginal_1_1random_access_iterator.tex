\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE, typename ALLOC$>$\newline
class original\+::random\+Access\+Iterator$<$ TYPE, ALLOC $>$}
Abstract base class for random-\/access iterators. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Type of elements being iterated \\
\hline
{\em ALLOC} & Allocator type used by the parent container for memory management\\
\hline
\end{DoxyTemplParams}
Implements core functionality for iterators that support\+:
\begin{DoxyItemize}
\item Pointer arithmetic operations (+, +=, -\/, -\/=)
\item Bidirectional traversal
\item Position validity checking
\item Element access/modification
\end{DoxyItemize}

\begin{DoxyNote}{Note}
The ALLOC template parameter must match the allocator type of the parent container. It is used to ensure type compatibility between the iterator and its container.

Maintains three state markers\+:
\begin{DoxyItemize}
\item Raw pointer to the current element
\item Reference to parent container (which uses the ALLOC allocator)
\item Absolute position index 
\end{DoxyItemize}
\end{DoxyNote}
