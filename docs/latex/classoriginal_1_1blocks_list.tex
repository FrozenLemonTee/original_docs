\subsubsection*{template$<$typename TYPE$>$\newline
class original\+::blocks\+List$<$ TYPE $>$}
A block-\/based list implementation. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Type of elements stored in the \doxylink{classoriginal_1_1blocks_list}{blocks\+List}\\
\hline
\end{DoxyTemplParams}
The \doxylink{classoriginal_1_1blocks_list}{blocks\+List} class is a container that stores elements in blocks, which makes it similar to std\+::deque. The class provides operations for insertion, deletion, and accessing elements from both ends. The internal structure consists of blocks of a fixed size, and elements are efficiently managed across these blocks. It also provides bidirectional iteration.
