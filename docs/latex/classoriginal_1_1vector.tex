class original\+::vector$<$ TYPE $>$}
Dynamic array container with amortized constant time operations. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Element type stored in the vector\\
\hline
\end{DoxyTemplParams}
Features include\+:
\begin{DoxyItemize}
\item Auto-\/resizing with centered memory allocation
\item Random access via operator\mbox{[}\mbox{]} with bounds checking
\item Bidirectional element insertion/removal
\item STL-\/style iterator support
\item Initializer list and array conversion 
\end{DoxyItemize}
