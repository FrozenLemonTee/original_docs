\subsubsection*{template$<$typename TYPE, typename ALLOC = allocator$<$\+TYPE$>$$>$\newline
class original\+::vector$<$ TYPE, ALLOC $>$}
Dynamic array container with amortized constant time operations. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Element type stored in the vector \\
\hline
{\em ALLOC} & Allocator type for memory management (default\+: allocator$<$\+TYPE$>$)\\
\hline
\end{DoxyTemplParams}
Features include\+:
\begin{DoxyItemize}
\item Auto-\/resizing with centered memory allocation
\item Random access via operator\mbox{[}\mbox{]} with bounds checking
\item Bidirectional element insertion/removal
\item STL-\/style iterator support
\item Initializer list and array conversion
\end{DoxyItemize}

Memory management characteristics\+:
\begin{DoxyItemize}
\item Uses the provided allocator for all memory operations
\item Grows exponentially (2x) when capacity is exceeded
\item Maintains centered memory layout for efficient front/back operations
\item Allocator propagation follows C++ standard rules
\end{DoxyItemize}
