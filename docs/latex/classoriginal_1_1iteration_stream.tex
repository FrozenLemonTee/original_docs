class original\+::iteration\+Stream$<$ TYPE, DERIVED $>$}
A stream class that allows iteration, comparison, and printing. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & The type of elements contained in the stream. \\
\hline
{\em DERIVED} & The derived class type.\\
\hline
\end{DoxyTemplParams}
This class allows iteration over a collection of elements of type {\ttfamily TYPE}. It provides methods for comparing two iteration streams and converting the stream into a string representation. The class is intended to be inherited by more specific stream types.
