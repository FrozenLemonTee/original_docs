\subsubsection*{template$<$typename DERIVED$>$\newline
class original\+::comparable$<$ DERIVED $>$}
Base class for comparable objects. 


\begin{DoxyTemplParams}{Template Parameters}
{\em DERIVED} & The type of the derived class.\\
\hline
\end{DoxyTemplParams}
This class defines a {\ttfamily \doxylink{classoriginal_1_1comparable_a2d23e08b034cda29c2854121f3e700d7}{compare\+To()}} method that must be implemented by derived classes to compare their instances. It also provides common comparison operators ({\ttfamily ==}, {\ttfamily !=}, {\ttfamily \texorpdfstring{$<$}{<}}, {\ttfamily \texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}=}, {\ttfamily \texorpdfstring{$>$}{>}=}) based on the {\ttfamily \doxylink{classoriginal_1_1comparable_a2d23e08b034cda29c2854121f3e700d7}{compare\+To()}} method.

Derived classes are expected to override {\ttfamily \doxylink{classoriginal_1_1comparable_a2d23e08b034cda29c2854121f3e700d7}{compare\+To()}} to return\+:
\begin{DoxyItemize}
\item A negative value if the current object is less than the other.
\item Zero if the current object is equal to the other.
\item A positive value if the current object is greater than the other. 
\end{DoxyItemize}
