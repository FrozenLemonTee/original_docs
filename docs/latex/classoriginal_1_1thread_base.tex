\subsubsection*{template$<$typename DERIVED$>$\newline
class original\+::thread\+Base$<$ DERIVED $>$}
Base class for thread implementations. 

Provides common thread functionality and interface

Key Features\+:
\begin{DoxyItemize}
\item Manages thread joinable state
\item Provides basic thread validity checks
\item Non-\/copyable but movable
\item Serves as base for both \doxylink{classoriginal_1_1p_thread}{p\+Thread} and thread implementations
\end{DoxyItemize}

\begin{DoxyNote}{Note}
This is an abstract base class and cannot be instantiated directly 
\end{DoxyNote}
