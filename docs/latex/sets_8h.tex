\doxysubsection{Detailed Description}
Implementation of set containers. 

Provides three set implementations with different underlying data structures\+:
\begin{DoxyEnumerate}
\item hash\+Set -\/ Hash table based implementation
\item tree\+Set -\/ Red-\/\+Black Tree based implementation
\item JSet -\/ Skip List based implementation
\end{DoxyEnumerate}

Common Features\+:
\begin{DoxyItemize}
\item Unique element storage (no duplicates)
\item Full iterator support
\item Customizable comparison/hash functions
\item Customizable allocators
\item Exception safety guarantees
\item Polymorphic usage through set interface
\end{DoxyItemize}
