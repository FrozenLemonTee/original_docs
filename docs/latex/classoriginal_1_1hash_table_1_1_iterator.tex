\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename K\+\_\+\+TYPE, typename V\+\_\+\+TYPE, typename ALLOC = allocator$<$\+K\+\_\+\+TYPE$>$, typename HASH = hash$<$\+K\+\_\+\+TYPE$>$$>$\newline
class original\+::hash\+Table$<$ K\+\_\+\+TYPE, V\+\_\+\+TYPE, ALLOC, HASH $>$\+::\+Iterator}
Forward iterator for \doxylink{classoriginal_1_1hash_table}{hash\+Table}. 

Provides forward iteration over \doxylink{classoriginal_1_1hash_table}{hash\+Table} elements with\+:
\begin{DoxyItemize}
\item Consistent traversal order (bucket by bucket, then chain order)
\item Safe invalidation detection
\item Const-\/correct access to elements
\end{DoxyItemize}

\doxylink{classoriginal_1_1hash_table_1_1_iterator}{Iterator} Characteristics\+:
\begin{DoxyItemize}
\item Forward iteration only (throws on reverse operations)
\item Invalidates on rehash operations
\item Lightweight copy semantics
\item STL-\/style iteration interface
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Iterators remain valid unless the table is rehashed 
\end{DoxyNote}
