\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE$>$\newline
class original\+::filter\+Stream$<$ TYPE $>$}
Composite filter builder with logical operator chaining. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Element type for filtering\\
\hline
\end{DoxyTemplParams}
Enables construction of complex filter conditions through operator overloading. Maintains internal operator precedence using postfix conversion. Use \doxylink{classoriginal_1_1filterStream_abcf25c575eb652dff5694bc1d07809e2}{group()} for explicit precedence control instead of parentheses.
