\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename K\+\_\+\+TYPE, typename V\+\_\+\+TYPE, typename HASH = hash$<$\+K\+\_\+\+TYPE$>$, typename ALLOC = allocator$<$couple$<$const K\+\_\+\+TYPE, V\+\_\+\+TYPE$>$$>$$>$\newline
class original\+::hash\+Map$<$ K\+\_\+\+TYPE, V\+\_\+\+TYPE, HASH, ALLOC $>$}
Hash table based implementation of the map interface. 


\begin{DoxyTemplParams}{Template Parameters}
{\em K\+\_\+\+TYPE} & Key type (must be hashable) \\
\hline
{\em V\+\_\+\+TYPE} & Value type \\
\hline
{\em HASH} & Hash function type (default\+: \doxylink{classoriginal_1_1hash}{hash$<$\+K\+\_\+\+TYPE$>$}) \\
\hline
{\em ALLOC} & Allocator type (default\+: allocator)\\
\hline
\end{DoxyTemplParams}
This class provides a concrete implementation of the map interface using a hash table with separate chaining. It combines the functionality of\+:
\begin{DoxyItemize}
\item map (interface)
\item \doxylink{classoriginal_1_1hash_table}{hash\+Table} (storage)
\item iterable (iteration support)
\end{DoxyItemize}

Performance Characteristics\+:
\begin{DoxyItemize}
\item Insertion\+: Average O(1), Worst O(n)
\item Lookup\+: Average O(1), Worst O(n)
\item Deletion\+: Average O(1), Worst O(n)
\end{DoxyItemize}

The implementation guarantees\+:
\begin{DoxyItemize}
\item Unique keys (no duplicates)
\item Type safety
\item Exception safety (basic guarantee)
\item \doxylink{classoriginal_1_1hash_map_1_1_iterator}{Iterator} validity unless modified 
\end{DoxyItemize}
