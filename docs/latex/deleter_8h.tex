\doxysubsection{Detailed Description}
Default deleters for resource management. 

Provides template classes for safely deleting single objects and arrays. Designed to be used with smart pointer implementations for automatic resource cleanup.

Key Features\+:
\begin{DoxyItemize}
\item Type-\/safe deletion policies for single objects and arrays
\item CRTP (Curiously Recurring Template Pattern) design for extensibility
\item Exception-\/safe operations with noexcept guarantees
\item Support for custom deletion policies through inheritance 
\end{DoxyItemize}
