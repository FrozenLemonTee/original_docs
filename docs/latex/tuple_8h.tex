\doxysubsection{Detailed Description}
Heterogeneous tuple container implementation. 

This file implements a type-\/safe, compile-\/time fixed-\/size tuple container with\+:
\begin{DoxyItemize}
\item Value semantics
\item Deep copy/move operations
\item Index-\/based element access
\item Lexicographical comparison
\item String serialization
\end{DoxyItemize}

\begin{DoxyNote}{Note}
The implementation uses recursive template instantiation for element storage 
\end{DoxyNote}
