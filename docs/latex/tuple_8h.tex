\doxysubsection{Detailed Description}
Heterogeneous tuple container implementation. 

This file implements a type-\/safe, compile-\/time fixed-\/size tuple container with\+:
\begin{DoxyItemize}
\item Element access by index
\item Element modification
\item Equality comparison
\item Formatted output
\item Move semantics
\item Structured binding support
\item Tuple concatenation and slicing
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Structured binding is enabled via {\ttfamily std\+::tuple\+\_\+size} and {\ttfamily std\+::tuple\+\_\+element} specializations. Value type deduction is used to ensure compatibility with {\ttfamily const auto\& \mbox{[}a, b, c\mbox{]} = my\+Tuple} syntax. 
\end{DoxyNote}
