\doxysubsection{Detailed Description}
Defines the iterator class for traversing and manipulating container elements. 

Provides the base class for iterators, implementing common operations such as dereferencing, comparison, cloning, and navigating through elements. Derived classes must implement specific behavior such as how to move to the next or previous element, and how to get and set the element\textquotesingle{}s value.
