\doxysubsection{Detailed Description}
Provides a generic hashing utility and interface for hashable types. 

This header defines\+:
\begin{DoxyItemize}
\item A comprehensive hash function object (hash) with specializations for common types
\item A base interface (hashable) for user-\/defined hashable types
\item Concept checking for hashable types (is\+Hashable)
\item Integration with std\+::hash for STL compatibility
\end{DoxyItemize}

Features\+:
\begin{DoxyItemize}
\item Implements FNV-\/1a algorithm for byte data
\item Specializations for integral types, pointers, strings, and custom types
\item Fallback implementations for trivially copyable types
\item Safe nullptr handling
\item Hash combination utilities
\end{DoxyItemize}

\begin{DoxyNote}{Note}
All hash functions are noexcept and provide basic hash distribution 
\end{DoxyNote}
