\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE, typename ALLOC = allocator$<$\+TYPE$>$$>$\newline
class original\+::base\+Array$<$ TYPE, ALLOC $>$}
Base class for fixed-\/size serial containers. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Type of the elements contained in the base array \\
\hline
{\em ALLOC} & Allocator type for memory management (default\+: allocator$<$\+TYPE$>$)\\
\hline
\end{DoxyTemplParams}
This class is used as the foundation for containers with a fixed size, such as static arrays. It inherits from \doxylink{classoriginal_1_1serial}{original\+::serial} and provides the standard serial container functionality while maintaining a constant size throughout its existence.

The allocator is used to manage the fixed-\/size memory block for the array elements. Unlike dynamic containers, the allocation occurs only once during construction and remains fixed throughout the container\textquotesingle{}s lifetime.

Key characteristics\+:
\begin{DoxyItemize}
\item Single allocation during construction
\item No reallocation operations
\item Fixed capacity matching the size
\item Allocator propagated to derived classes
\end{DoxyItemize}

The ALLOC type must meet the C++ allocator requirements and should provide\+:

\item propagate\+\_\+on\+\_\+container\+\_\+copy\+\_\+assignment/move\+\_\+assignment typedefs 
\end{DoxyItemize}
