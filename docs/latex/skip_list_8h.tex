\doxysubsection{Detailed Description}
Skip List container implementation. 

Provides a probabilistic alternative to balanced trees with\+:
\begin{DoxyItemize}
\item Expected O(log n) search/insert/delete operations
\item Multi-\/level linked list structure
\item Custom comparator support
\item STL-\/style allocator support
\end{DoxyItemize}

Key Features\+:
\begin{DoxyItemize}
\item Probabilistic balancing with O(log n) expected performance
\item Simpler implementation than balanced trees
\item Sorted element storage
\item Customizable comparison and allocation
\item Full iterator support
\item Exception safety (basic guarantee) 
\end{DoxyItemize}
