\doxysubsection{Detailed Description}
Thread management utilities. 

Provides a layered threading abstraction with\+:
\begin{DoxyItemize}
\item Low-\/level POSIX thread wrapper (p\+Thread)
\item High-\/level RAII thread management (thread)
\item Exception-\/safe thread operations
\item Flexible join/detach policies
\end{DoxyItemize}

Key Features\+:
\begin{DoxyItemize}
\item Three-\/layer thread abstraction\+:
\begin{DoxyEnumerate}
\item thread\+Base -\/ Common interface and state management
\item p\+Thread -\/ POSIX thread wrapper
\item thread -\/ High-\/level RAII wrapper
\end{DoxyEnumerate}
\item Automatic resource management
\item Move-\/only semantics
\item Exception safety (basic guarantee)
\item Configurable join/detach behavior
\end{DoxyItemize}

Thread Safety\+:
\begin{DoxyItemize}
\item Individual thread objects are not thread-\/safe
\item External synchronization required for shared access
\end{DoxyItemize}

Exception Handling\+:
\begin{DoxyItemize}
\item Throws sys\+Error for thread operation failures
\item Destructors may throw if thread is joinable and not joined/detached 
\end{DoxyItemize}
