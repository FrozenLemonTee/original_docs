An iterator adapter for the iterable container. 

The {\ttfamily \doxylink{classoriginal_1_1iterable_1_1iter_adaptor}{iter\+Adaptor}} class is an implementation of the iterator interface that adapts the base iterator ({\ttfamily \doxylink{classoriginal_1_1base_iterator}{base\+Iterator}\texorpdfstring{$<$}{<}TYPE\texorpdfstring{$>$}{>}}) and provides additional iterator functionalities to the {\ttfamily iterable} class.
