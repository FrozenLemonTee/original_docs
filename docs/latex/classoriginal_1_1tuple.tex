\subsubsection*{template$<$typename... TYPES$>$\newline
class original\+::tuple$<$ TYPES $>$}
Container for multiple heterogeneous elements. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPES} & Variadic template parameter list of element types\\
\hline
\end{DoxyTemplParams}
Stores a sequence of elements with type safety. Provides\+:
\begin{DoxyItemize}
\item Compile-\/time fixed size container
\item Element access via index (compile-\/time checked)
\item Element modification via index
\item Deep copy/move operations
\item Lexicographical comparison
\item String serialization through printable interface
\item \doxylink{conceptoriginal_1_1Comparable}{Comparable} interface implementation
\item Structured binding support via std\+::tuple\+\_\+size and std\+::tuple\+\_\+element specializations
\item Tuple concatenation and slicing operations 
\end{DoxyItemize}\begin{Desc}
