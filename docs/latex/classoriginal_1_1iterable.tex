\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE$>$\newline
class original\+::iterable$<$ TYPE $>$}
A base class for iterable containers that support multiple iteration patterns. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & The type of elements contained in the iterable.\\
\hline
\end{DoxyTemplParams}
This class defines the complete iterable container interface with\+:
\begin{DoxyItemize}
\item Polymorphic iterator factories (\doxylink{classoriginal_1_1iterable_a82918c322a679ddad8897440f29bc5bc}{begins()}/ \doxylink{classoriginal_1_1iterable_af986250dbca5b4ef7c0a60bb4d183e70}{ends()})
\item Standard library compatible iteration (\doxylink{classoriginal_1_1iterable_abbe2b7be21e8c971b9b548c46d394014}{begin()}/ \doxylink{classoriginal_1_1iterable_a573f7bd567b302868762aaeff0f7de9b}{end()})
\item Direct element access (\doxylink{classoriginal_1_1iterable_a35d7da844c6d25bc1a24b7fc1583aa80}{first()}/ \doxylink{classoriginal_1_1iterable_ae26473236643fba6b1dc52143498fdb4}{last()})
\item RAII-\/managed iterator adapters (\doxylink{classoriginal_1_1iterable_1_1iter_adaptor}{iter\+Adaptor})
\end{DoxyItemize}

The class provides three levels of iteration interface\+:
\begin{DoxyEnumerate}
\item Low-\/level\+: \doxylink{classoriginal_1_1iterable_a82918c322a679ddad8897440f29bc5bc}{begins()}/ \doxylink{classoriginal_1_1iterable_af986250dbca5b4ef7c0a60bb4d183e70}{ends()} -\/ Factory methods returning raw base iterators
\item Mid-\/level\+: \doxylink{classoriginal_1_1iterable_a35d7da844c6d25bc1a24b7fc1583aa80}{first()}/ \doxylink{classoriginal_1_1iterable_ae26473236643fba6b1dc52143498fdb4}{last()} -\/ RAII-\/wrapped direct element access
\item High-\/level\+: \doxylink{classoriginal_1_1iterable_abbe2b7be21e8c971b9b548c46d394014}{begin()}/ \doxylink{classoriginal_1_1iterable_a573f7bd567b302868762aaeff0f7de9b}{end()} -\/ STL-\/compatible range iteration
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
Derived classes must implement \doxylink{classoriginal_1_1iterable_a82918c322a679ddad8897440f29bc5bc}{begins()}/ends() to return their specific iterator type (covariant return types supported). All other iteration methods build upon these primitives.
\end{DoxyNote}
