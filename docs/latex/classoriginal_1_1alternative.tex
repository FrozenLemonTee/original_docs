\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE$>$\newline
class original\+::alternative$<$ TYPE $>$}
A type-\/safe container that may or may not contain a value. 

Specialization for void type alternative.


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & The type of value to store\\
\hline
\end{DoxyTemplParams}
This class provides a way to represent optional values without using pointers. It can either contain a value of type TYPE or be empty. The empty state is represented using \doxylink{classoriginal_1_1none}{original\+::none} from \doxylink{types_8h}{types.\+h}.

Example usage\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{classoriginal_1_1alternative_a2dbc3c0a86563dcd3c65ba888722ebc6}{alternative<int>}}\ opt;\ \ \ \ \ \ \ \ \ \ \textcolor{comment}{//\ Empty\ state}}
\DoxyCodeLine{\mbox{\hyperlink{classoriginal_1_1alternative_a2dbc3c0a86563dcd3c65ba888722ebc6}{alternative<int>}}\ opt2(42);\ \ \ \ \ \textcolor{comment}{//\ Contains\ value\ 42}}
\DoxyCodeLine{\textcolor{keywordflow}{if}\ (opt2)\ \{\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \textcolor{comment}{//\ Check\ if\ has\ value}}
\DoxyCodeLine{\ \ \ \ \textcolor{keywordtype}{int}\ val\ =\ *opt2;\ \ \ \ \ \ \ \ \ \ \ \textcolor{comment}{//\ Access\ the\ value}}
\DoxyCodeLine{\}}

\end{DoxyCode}


Represents a simple boolean flag indicating presence or absence of a value without storing actual data. Useful for signaling states without associated data.
