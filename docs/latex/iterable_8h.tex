\doxysubsection{Detailed Description}
Base interface for iterable container types. 

Defines the core abstraction for iterator-\/enabled containers. Provides\+:
\begin{DoxyItemize}
\item iterable$<$\+TYPE$>$ base class with iterator access methods
\item iter\+Adaptor nested class implementing iterator pattern
\item Range-\/based for loop support via begin()/end() methods
\item Element traversal and manipulation operations
\end{DoxyItemize}
