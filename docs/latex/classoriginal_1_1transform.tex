\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE$>$\newline
class original\+::transform$<$ TYPE $>$}
Base class for transformation operations. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & The type of element being transformed\\
\hline
\end{DoxyTemplParams}
This class provides an interface for transformations that apply operations to elements of type {\ttfamily TYPE}. Derived classes must implement the {\ttfamily apply} method to perform the specific transformation. The {\ttfamily operator()} allows the transformation to be applied by invoking the {\ttfamily apply} method.
