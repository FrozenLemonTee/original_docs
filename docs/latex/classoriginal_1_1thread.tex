\doxysubsection{Detailed Description}
High-\/level thread wrapper. 

Manages thread lifetime with automatic join/detach. Provides RAII semantics for thread management with configurable join policy. Implements \doxylink{classoriginal_1_1thread_base}{thread\+Base} interface while wrapping a \doxylink{classoriginal_1_1p_thread}{p\+Thread} instance.

Key Features\+:
\begin{DoxyItemize}
\item Wraps low-\/level \doxylink{classoriginal_1_1p_thread}{p\+Thread} with automatic cleanup
\item Configurable join policy (AUTO\+\_\+\+JOIN or AUTO\+\_\+\+DETACH)
\item Implements \doxylink{classoriginal_1_1thread_base}{thread\+Base} interface
\item Delegates all thread operations to contained \doxylink{classoriginal_1_1p_thread}{p\+Thread} instance
\end{DoxyItemize}

Join Policy\+:
\begin{DoxyItemize}
\item join\+Policy\+::\+AUTO\+\_\+\+JOIN\+: join the thread in destructor
\item join\+Policy\+::\+AUTO\+\_\+\+DETACH\+: detach the thread in destructor
\end{DoxyItemize}
