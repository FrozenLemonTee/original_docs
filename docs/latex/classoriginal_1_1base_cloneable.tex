\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename DERIVED$>$\newline
class original\+::base\+Cloneable$<$ DERIVED $>$}
Base class that defines a cloneable interface. 


\begin{DoxyTemplParams}{Template Parameters}
{\em DERIVED} & The type of the derived class.\\
\hline
\end{DoxyTemplParams}
This class provides the mechanism to clone derived objects by calling {\ttfamily \doxylink{classoriginal_1_1base_cloneable_abe3520c8dc2f0b77219b0ec620412b13}{clone()}} in derived classes. It is designed for situations where the cloning behavior does not need to be customized. Derived classes can simply use the default implementation of {\ttfamily \doxylink{classoriginal_1_1base_cloneable_abe3520c8dc2f0b77219b0ec620412b13}{clone()}} provided by this class.
