class original\+::deleter\+Base$<$ TYPE, DERIVED $>$}
Base class for deleters defining the deletion policy interface. 

Base class for array deleters.


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Type of the resource. \\
\hline
{\em DERIVED} & CRTP Derived deleter type.\\
\hline
\end{DoxyTemplParams}
This class serves as the base for all specific deleters and provides a pure virtual function {\ttfamily operator()} to handle resource deletion. Since this is an abstract class, it cannot be instantiated directly.

\begin{DoxyNote}{Note}
Inherit from this class to create custom deletion policies.
\end{DoxyNote}
