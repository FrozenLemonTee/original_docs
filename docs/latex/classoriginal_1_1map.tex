\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename K\+\_\+\+TYPE, typename V\+\_\+\+TYPE, typename ALLOC = allocator$<$couple$<$const K\+\_\+\+TYPE, V\+\_\+\+TYPE$>$$>$$>$\newline
class original\+::map$<$ K\+\_\+\+TYPE, V\+\_\+\+TYPE, ALLOC $>$}
Abstract base class for key-\/value mapping containers. 


\begin{DoxyTemplParams}{Template Parameters}
{\em K\+\_\+\+TYPE} & The key type (must support equality comparison and hashing) \\
\hline
{\em V\+\_\+\+TYPE} & The value type to store \\
\hline
{\em ALLOC} & Allocator type for memory management (default\+: allocator)\\
\hline
\end{DoxyTemplParams}
This abstract class defines the common interface for all map implementations in the framework. Concrete implementations should inherit from this class and provide the actual storage mechanism.

The map guarantees\+:
\begin{DoxyItemize}
\item Unique keys (no duplicates)
\item Efficient value lookup by key
\item Type-\/safe operations
\item Exception safety (throws \doxylink{classoriginal_1_1no_element_error}{no\+Element\+Error} for invalid access)
\end{DoxyItemize}

\begin{DoxyNote}{Note}
This class is not meant to be instantiated directly -\/ use concrete implementations like \doxylink{classoriginal_1_1hash_map}{hash\+Map} or tree\+Map instead. 
\end{DoxyNote}
