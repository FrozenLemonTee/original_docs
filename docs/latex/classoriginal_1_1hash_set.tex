\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE, typename HASH = hash$<$\+TYPE$>$, typename ALLOC = allocator$<$couple$<$const TYPE, const bool$>$$>$$>$\newline
class original\+::hash\+Set$<$ TYPE, HASH, ALLOC $>$}
Hash table based implementation of the set interface. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Element type (must be hashable) \\
\hline
{\em HASH} & Hash function type (default\+: hash$<$\+TYPE$>$) \\
\hline
{\em ALLOC} & Allocator type (default\+: allocator$<$couple$<$const TYPE, const bool$>$$>$)\\
\hline
\end{DoxyTemplParams}
This class provides a concrete implementation of the set interface using a hash table with separate chaining. It combines the functionality of\+:
\begin{DoxyItemize}
\item set (interface)
\item \doxylink{classoriginal_1_1hash_table}{hash\+Table} (storage with bool values)
\item iterable (iteration support)
\end{DoxyItemize}

Performance Characteristics\+:
\begin{DoxyItemize}
\item Insertion\+: Average O(1), Worst O(n)
\item Lookup\+: Average O(1), Worst O(n)
\item Deletion\+: Average O(1), Worst O(n)
\end{DoxyItemize}

The implementation guarantees\+:
\begin{DoxyItemize}
\item Unique elements (no duplicates)
\item Type safety
\item Exception safety (basic guarantee)
\item \doxylink{classoriginal_1_1hash_set_1_1_iterator}{Iterator} validity unless modified 
\end{DoxyItemize}
