\doxysubsection{Detailed Description}
Utility class containing generic container algorithms. 

Provides static methods for iterator-\/based container operations including\+:
\begin{DoxyItemize}
\item Range queries (distance, front/back navigation)
\item Element search/count (find, count, condition checks)
\item Data manipulation (fill, swap, copy, reverse)
\item Heap operations (heapify, adjust, priority comparison)
\item Conditional element processing
\item Generic sorting algorithm 
\end{DoxyItemize}
