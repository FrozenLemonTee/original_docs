\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE, template$<$ typename, typename $>$ typename SERIAL = chain, template$<$ typename $>$ typename ALLOC = allocator$>$\newline
class original\+::deque$<$ TYPE, SERIAL, ALLOC $>$}
Double-\/ended queue container adapter. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Type of elements stored in the deque \\
\hline
{\em SERIAL} & Underlying container type (default\+: chain) \\
\hline
{\em ALLOC} & Allocator template for memory management (default\+: allocator)\\
\hline
\end{DoxyTemplParams}
Implements deque operations using the specified underlying container. Supports efficient insertion and removal at both front and back ends. Inherits template constraints from \doxylink{classoriginal_1_1containerAdapter}{original\+::container\+Adapter}.

The allocator is propagated to both the deque adapter and the underlying serial container for consistent memory management.
