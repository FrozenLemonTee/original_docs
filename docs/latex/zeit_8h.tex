\doxysubsection{Detailed Description}
Time and date handling utilities. 

Provides classes for time duration, points in time, and UTC date/time handling. Includes support for\+:
\begin{DoxyItemize}
\item Time durations with various units (nanoseconds to days)
\item Time points representing moments in time
\item UTC date/time with calendar operations
\item Literals for time durations
\end{DoxyItemize}

Key features\+:
\begin{DoxyItemize}
\item High precision time representation (nanoseconds)
\item Calendar calculations (leap years, days in month, etc.)
\item Platform-\/independent time operations
\item Comparable, hashable, and printable interfaces 
\end{DoxyItemize}
