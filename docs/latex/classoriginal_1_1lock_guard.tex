\doxysubsection{Detailed Description}
Abstract base class for lock guard implementations. 

Provides the interface for RAII-\/style lock management including\+:
\begin{DoxyItemize}
\item Various locking policies (automatic, manual, try-\/lock)
\item Lock/unlock functionality
\item Lock state query 
\end{DoxyItemize}
