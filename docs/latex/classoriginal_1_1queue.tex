\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE, template$<$ typename, typename $>$ typename SERIAL = chain, template$<$ typename $>$ typename ALLOC = allocator$>$\newline
class original\+::queue$<$ TYPE, SERIAL, ALLOC $>$}
First-\/\+In-\/\+First-\/\+Out (FIFO) container adapter. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Type of elements stored in the queue \\
\hline
{\em SERIAL} & Underlying container type (default\+: chain) \\
\hline
{\em ALLOC} & Allocator type for memory management (default\+: allocator$<$\+TYPE$>$)\\
\hline
\end{DoxyTemplParams}
Implements queue operations using the specified underlying container. Supports insertion at back and removal from front. Inherits template constraints from \doxylink{classoriginal_1_1container_adapter}{original\+::container\+Adapter}.

The allocator is propagated to both the queue adapter and the underlying serial container for consistent memory management. The ALLOC type must meet the C++ allocator requirements.
