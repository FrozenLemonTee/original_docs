\subsubsection*{template$<$typename TYPE$>$\newline
class original\+::hash$<$ TYPE $>$}
Forward declaration of hash class template. 

Generic hash function object supporting multiple types.


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & The type for which the hash function object is defined.\\
\hline
{\em TYPE} & The type for which the hash function object is defined\\
\hline
\end{DoxyTemplParams}
Provides\+:
\begin{DoxyItemize}
\item FNV-\/1a hash algorithm implementation
\item Specialized hash functions for common types
\item Fallback implementations for arbitrary types
\item Consistent hashing interface via operator()
\end{DoxyItemize}

Supported Types\+:
\begin{DoxyItemize}
\item Integral types (direct casting)
\item Pointers (address-\/based hashing)
\item Strings (FNV-\/1a over characters)
\item Trivially copyable types (byte-\/wise hashing)
\item Types implementing hashable interface 
\end{DoxyItemize}
