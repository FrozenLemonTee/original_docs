\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE, typename ALLOC$>$\newline
class original\+::serial$<$ TYPE, ALLOC $>$}
Abstract base class for sequential containers with index-\/based access. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Type of the elements contained in the serial container \\
\hline
{\em ALLOC} & Allocator type for memory management (default\+: depends on derived class)\\
\hline
\end{DoxyTemplParams}
Defines common interface for containers that support index-\/based access to their elements. It provides methods for element retrieval, bounds checking, and negative index handling. Derived classes must implement {\ttfamily \doxylink{classoriginal_1_1serial_a6e5580f8ff611f5dbf73c78b5a157da7}{get()}}, {\ttfamily operator\mbox{[}\mbox{]}}, {\ttfamily \doxylink{classoriginal_1_1serial_a7ffffd44b78db8306485015c254074ad}{set()}}, and {\ttfamily \doxylink{classoriginal_1_1serial_a9fd361e7f0b340e17cb6b1b190013d14}{index\+Of()}} methods.

The allocator is inherited from the base container class and is propagated to derived classes for consistent memory management. The ALLOC type must meet the C++ allocator requirements.
