\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE$>$\newline
class original\+::serial$<$ TYPE $>$}
Abstract base class for sequential containers with index-\/based access. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Type of the elements contained in the serial container\\
\hline
\end{DoxyTemplParams}
Defines common interface for containers that support index-\/based access to their elements. It provides methods for element retrieval, bounds checking, and negative index handling. Derived classes must implement {\ttfamily \doxylink{classoriginal_1_1serial_a78abe0a415dd59ddd648e8a028051fca}{get()}}, {\ttfamily operator\mbox{[}\mbox{]}}, {\ttfamily \doxylink{classoriginal_1_1serial_a5a6aeda0aff18e4596fbec8295b81adb}{set()}}, and {\ttfamily \doxylink{classoriginal_1_1serial_a52d30588c2517ea4fc4ba5bce5189cc1}{index\+Of()}} methods.
