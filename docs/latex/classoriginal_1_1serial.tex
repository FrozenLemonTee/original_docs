\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE$>$\newline
class original\+::serial$<$ TYPE $>$}
Abstract base class for sequential containers with index-\/based access. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Type of the elements contained in the serial container\\
\hline
\end{DoxyTemplParams}
Defines common interface for containers that support index-\/based access to their elements. It provides methods for element retrieval, bounds checking, and negative index handling. Derived classes must implement {\ttfamily \doxylink{classoriginal_1_1serial_a7bec66c89d86eae6ee6eebef3d09fe6e}{get()}}, {\ttfamily operator\mbox{[}\mbox{]}}, {\ttfamily \doxylink{classoriginal_1_1serial_a5aa2cefd30960487587987b884edf491}{set()}}, and {\ttfamily \doxylink{classoriginal_1_1serial_a13d19a465b96a003bce55edd73b2b1df}{index\+Of()}} methods.
