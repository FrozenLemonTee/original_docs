\doxysubsection{Detailed Description}
Represents a UTC calendar date and time. 

Provides calendar operations including\+:
\begin{DoxyItemize}
\item Leap year detection
\item Days in month calculation
\item Weekday calculation
\item Conversion to/from time points 
\end{DoxyItemize}
