\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename K\+\_\+\+TYPE, typename V\+\_\+\+TYPE, typename Compare = increase\+Comparator$<$\+K\+\_\+\+TYPE$>$, typename ALLOC = allocator$<$couple$<$const K\+\_\+\+TYPE, V\+\_\+\+TYPE$>$$>$$>$\newline
class original\+::\+JMap$<$ K\+\_\+\+TYPE, V\+\_\+\+TYPE, Compare, ALLOC $>$}
Skip List based implementation of the map interface. 


\begin{DoxyTemplParams}{Template Parameters}
{\em K\+\_\+\+TYPE} & Key type (must be comparable) \\
\hline
{\em V\+\_\+\+TYPE} & Value type \\
\hline
{\em \doxylink{conceptoriginal_1_1_compare}{Compare}} & Comparison function type (default\+: \doxylink{classoriginal_1_1increase_comparator}{increase\+Comparator$<$\+K\+\_\+\+TYPE$>$}) \\
\hline
{\em ALLOC} & Allocator type (default\+: allocator$<$couple$<$const K\+\_\+\+TYPE, V\+\_\+\+TYPE$>$$>$)\\
\hline
\end{DoxyTemplParams}
This class provides a concrete implementation of the map interface using a probabilistic skip list. It combines the functionality of\+:
\begin{DoxyItemize}
\item map (interface)
\item \doxylink{classoriginal_1_1skip_list}{skip\+List} (storage)
\item iterable (iteration support)
\end{DoxyItemize}

Performance Characteristics\+:
\begin{DoxyItemize}
\item Insertion\+: Average O(log n), Worst O(n)
\item Lookup\+: Average O(log n), Worst O(n)
\item Deletion\+: Average O(log n), Worst O(n)
\end{DoxyItemize}

The implementation guarantees\+:
\begin{DoxyItemize}
\item Elements sorted by key according to comparator
\item Unique keys (no duplicates)
\item Type safety
\item Exception safety (basic guarantee)
\item \doxylink{classoriginal_1_1_j_map_1_1_iterator}{Iterator} validity unless modified 
\end{DoxyItemize}
