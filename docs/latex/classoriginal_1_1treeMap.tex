\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename K\+\_\+\+TYPE, typename V\+\_\+\+TYPE, typename Compare = increase\+Comparator$<$\+K\+\_\+\+TYPE$>$, typename ALLOC = allocator$<$couple$<$const K\+\_\+\+TYPE, V\+\_\+\+TYPE$>$$>$$>$\newline
class original\+::tree\+Map$<$ K\+\_\+\+TYPE, V\+\_\+\+TYPE, Compare, ALLOC $>$}
Red-\/\+Black Tree based implementation of the map interface. 


\begin{DoxyTemplParams}{Template Parameters}
{\em K\+\_\+\+TYPE} & Key type (must be comparable) \\
\hline
{\em V\+\_\+\+TYPE} & Value type \\
\hline
{\em \doxylink{conceptoriginal_1_1Compare}{Compare}} & Comparison function type (default\+: \doxylink{classoriginal_1_1increaseComparator}{increase\+Comparator$<$\+K\+\_\+\+TYPE$>$}) \\
\hline
{\em ALLOC} & Allocator type (default\+: allocator)\\
\hline
\end{DoxyTemplParams}
This class provides a concrete implementation of the map interface using a Red-\/\+Black Tree. It combines the functionality of\+:
\begin{DoxyItemize}
\item map (interface)
\item \doxylink{classoriginal_1_1RBTree}{RBTree} (storage)
\item iterable (iteration support)
\end{DoxyItemize}

Performance Characteristics\+:
\begin{DoxyItemize}
\item Insertion\+: O(log n)
\item Lookup\+: O(log n)
\item Deletion\+: O(log n)
\item Traversal\+: O(n)
\end{DoxyItemize}

The implementation guarantees\+:
\begin{DoxyItemize}
\item Elements sorted by key according to comparator
\item Unique keys (no duplicates)
\item Type safety
\item Exception safety (basic guarantee)
\item \doxylink{classoriginal_1_1treeMap_1_1Iterator}{Iterator} validity unless modified 
\end{DoxyItemize}
