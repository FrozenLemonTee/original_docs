\doxysubsection{Detailed Description}
Memory allocation interface and implementations. 

Provides\+:
\begin{DoxyItemize}
\item Base allocator interface with CRTP support (allocator\+Base)
\item Utility functions for raw memory operations (allocators class)
\item Default memory allocator implementation (allocator)
\item Object pool allocator implementation (obj\+Pool\+Allocator)
\end{DoxyItemize}
