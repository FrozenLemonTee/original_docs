\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE, typename Compare = increase\+Comparator$<$\+TYPE$>$, typename ALLOC = allocator$<$couple$<$const TYPE, const bool$>$$>$$>$\newline
class original\+::\+JSet$<$ TYPE, Compare, ALLOC $>$}
Skip List based implementation of the set interface. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Element type (must be comparable) \\
\hline
{\em \doxylink{conceptoriginal_1_1_compare}{Compare}} & Comparison function type (default\+: increase\+Comparator$<$\+TYPE$>$) \\
\hline
{\em ALLOC} & Allocator type (default\+: allocator$<$couple$<$const TYPE, const bool$>$$>$)\\
\hline
\end{DoxyTemplParams}
This class provides a concrete implementation of the set interface using a probabilistic skip list. It combines the functionality of\+:
\begin{DoxyItemize}
\item set (interface)
\item \doxylink{classoriginal_1_1skip_list}{skip\+List} (storage with bool values)
\item iterable (iteration support)
\end{DoxyItemize}

Performance Characteristics\+:
\begin{DoxyItemize}
\item Insertion\+: Average O(log n), Worst O(n)
\item Lookup\+: Average O(log n), Worst O(n)
\item Deletion\+: Average O(log n), Worst O(n)
\end{DoxyItemize}

The implementation guarantees\+:
\begin{DoxyItemize}
\item Elements sorted according to comparator
\item Unique elements (no duplicates)
\item Type safety
\item Exception safety (basic guarantee)
\item \doxylink{classoriginal_1_1_j_set_1_1_iterator}{Iterator} validity unless modified 
\end{DoxyItemize}
