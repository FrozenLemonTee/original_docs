\doxysubsection{Detailed Description}
Double-\/direction iterator base class. 

Provides common functionality for iterators that can only traverse in both directions (forward and backward). It extends the {\ttfamily step\+Iterator} class to include functionality that applies to single-\/direction step iterators.
