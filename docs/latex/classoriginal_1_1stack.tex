Last-\/\+In-\/\+First-\/\+Out (LIFO) container adapter \texorpdfstring{$<$}{<}TYPE, SERIAL\texorpdfstring{$>$}{>} 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Type of elements stored in the stack \\
\hline
{\em SERIAL} & Underlying container type (default\+: chain), see constraints from \doxylink{classoriginal_1_1container_adapter}{original\+::container\+Adapter}\\
\hline

Implements stack operations using the specified underlying container type. Provides standard stack interface including push, pop, top, and comparison operations. The adapter maintains elements through composition with the specified SERIAL container.

\end{DoxyNote}
