Base class for filter operations. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Type of the element to match\\
\hline
\end{DoxyTemplParams}
This class provides the interface for performing match operations on an element of type {\ttfamily TYPE}. Derived classes should implement the {\ttfamily match} method for specific matching logic. The {\ttfamily operator()} method is used to invoke the {\ttfamily match} method.
