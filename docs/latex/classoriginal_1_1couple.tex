Container for two heterogeneous elements. 


\begin{DoxyTemplParams}{Template Parameters}
{\em F\+\_\+\+TYPE} & Type of the first element \\
\hline
{\em S\+\_\+\+TYPE} & Type of the second element\\

Stores a pair of elements with type safety. Provides\+:
\begin{DoxyItemize}
\item Element access and copy operations
\item Move construction and assignment
\item Lexicographical comparison (first element precedence)
\item Formatted string output through printable interface
\item \doxylink{conceptoriginal_1_1_comparable}{Comparable} interface implementation
\item Structured binding support via std\+::tuple\+\_\+size and std\+::tuple\+\_\+element specializations 
\end{DoxyItemize}
