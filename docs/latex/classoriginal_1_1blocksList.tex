\subsubsection*{template$<$typename TYPE, typename ALLOC = allocator$<$\+TYPE$>$$>$\newline
class original\+::blocks\+List$<$ TYPE, ALLOC $>$}
A block-\/based list implementation. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Type of elements stored in the \doxylink{classoriginal_1_1blocksList}{blocks\+List} \\
\hline
{\em ALLOC} & Allocator type to use for memory management (default\+: \doxylink{classoriginal_1_1allocator}{allocator$<$\+TYPE$>$})\\
\hline
\end{DoxyTemplParams}
The \doxylink{classoriginal_1_1blocksList}{blocks\+List} class is a container that stores elements in blocks. The class provides operations for insertion, deletion, and accessing elements from both ends. Memory management is handled through the specified allocator type, which is used for both block allocation and element construction/destruction.
