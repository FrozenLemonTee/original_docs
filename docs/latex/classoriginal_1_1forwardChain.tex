class original\+::forward\+Chain$<$ TYPE, ALLOC $>$}
A singly linked list implementation. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & The type of elements stored in the forward chain \\
\hline
{\em ALLOC} & Allocator type for memory management (default\+: \doxylink{classoriginal_1_1allocator}{allocator$<$\+TYPE$>$})\\
\hline
\end{DoxyTemplParams}
The \doxylink{classoriginal_1_1forwardChain}{forward\+Chain} class implements a singly linked list where elements are stored in nodes. Each node points to the next node, and the list supports operations like push, pop, get, and index\+Of. Uses the provided allocator for all memory management operations, including node allocation.
