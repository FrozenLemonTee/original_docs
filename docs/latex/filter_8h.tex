\doxysubsection{Detailed Description}
Filter base class and derived filter classes for various matching operations. 

This file provides the definition of a base class {\ttfamily filter} for matching elements based on specific conditions, and derived classes that implement different types of filters, such as equality, inequality, range checking, etc. The filter classes include\+:
\begin{DoxyItemize}
\item {\ttfamily equal\+Filter}\+: checks if an element is equal to a target value.
\item {\ttfamily not\+Equal\+Filter}\+: checks if an element is not equal to a target value.
\item {\ttfamily less\+Filter}\+: checks if an element is less than a target value.
\item {\ttfamily greater\+Filter}\+: checks if an element is greater than a target value.
\item {\ttfamily not\+Less\+Filter}\+: checks if an element is greater than or equal to a target value.
\item {\ttfamily not\+Greater\+Filter}\+: checks if an element is less than or equal to a target value.
\item {\ttfamily range\+Filter}\+: checks if an element lies within a given range. 
\end{DoxyItemize}
