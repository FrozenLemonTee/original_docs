\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE, typename DELETER$>$\newline
class original\+::strong\+Ptr$<$ TYPE, DELETER $>$}
Shared ownership smart pointer with strong references. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Managed object type \\
\hline
{\em DELETER} & Deletion policy type (default\+: deleter$<$\+TYPE$>$)\\
\hline
\end{DoxyTemplParams}
Maintains object lifetime through reference counting\+:
\begin{DoxyItemize}
\item Increases strong count on copy
\item Decreases strong count on destruction
\item Destroys object when strong count reaches zero
\item Supports copy/move semantics for shared ownership 
\end{DoxyItemize}
