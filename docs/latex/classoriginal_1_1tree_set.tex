\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TYPE, typename Compare = increase\+Comparator$<$\+TYPE$>$, typename ALLOC = allocator$<$couple$<$const TYPE, const bool$>$$>$$>$\newline
class original\+::tree\+Set$<$ TYPE, Compare, ALLOC $>$}
Red-\/\+Black Tree based implementation of the set interface. 


\begin{DoxyTemplParams}{Template Parameters}
{\em TYPE} & Element type (must be comparable) \\
\hline
{\em \doxylink{conceptoriginal_1_1_compare}{Compare}} & Comparison function type (default\+: increase\+Comparator$<$\+TYPE$>$) \\
\hline
{\em ALLOC} & Allocator type (default\+: allocator$<$couple$<$const TYPE, const bool$>$$>$)\\
\hline
\end{DoxyTemplParams}
This class provides a concrete implementation of the set interface using a Red-\/\+Black Tree. It combines the functionality of\+:
\begin{DoxyItemize}
\item set (interface)
\item \doxylink{classoriginal_1_1_r_b_tree}{RBTree} (storage with bool values)
\item iterable (iteration support)
\end{DoxyItemize}

Performance Characteristics\+:
\begin{DoxyItemize}
\item Insertion\+: O(log n)
\item Lookup\+: O(log n)
\item Deletion\+: O(log n)
\item Traversal\+: O(n)
\end{DoxyItemize}

The implementation guarantees\+:
\begin{DoxyItemize}
\item Elements sorted according to comparator
\item Unique elements (no duplicates)
\item Type safety
\item Exception safety (basic guarantee)
\item \doxylink{classoriginal_1_1tree_set_1_1_iterator}{Iterator} validity unless modified 
\end{DoxyItemize}
