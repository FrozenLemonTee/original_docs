\doxysubsection{Detailed Description}
Base interface for all custom exception types. 

Inherits from std\+::exception to provide standard exception handling integration. Serves as the root type for all domain-\/specific exceptions in the library.

\begin{DoxyNote}{Note}
All custom exceptions should inherit from this class to maintain consistent error handling behavior across the codebase. 
\end{DoxyNote}
