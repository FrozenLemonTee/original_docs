\doxysubsection{Detailed Description}
Non-\/cyclic doubly linked list implementation. 

Provides a bidirectional linked list container with efficient front/back operations. Features include\+:
\begin{DoxyItemize}
\item Constant time insertion/removal at both ends
\item Linear time random access
\item Safe element ownership through RAII 
\end{DoxyItemize}
