Comparator base class and concrete comparator classes. 

This file provides a base comparator class and several derived classes for different types of comparisons\+:
\begin{DoxyItemize}
\item {\ttfamily increase\+Comparator}\+: for increasing comparisons (less than)
\item {\ttfamily decrease\+Comparator}\+: for decreasing comparisons (greater than)
\item {\ttfamily equal\+Comparator}\+: for equality comparisons (equal to)
\item {\ttfamily not\+Equal\+Comparator}\+: for inequality comparisons (not equal to)
\item {\ttfamily increase\+Not\+Strict\+Comparator}\+: for non-\/strict increasing comparisons (less than or equal to)
\item {\ttfamily decrease\+Not\+Strict\+Comparator}\+: for non-\/strict decreasing comparisons (greater than or equal to) 
\end{DoxyItemize}
