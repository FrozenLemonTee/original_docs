\doxysubsection{Detailed Description}
Implementation of map containers. 

Provides three map implementations with different underlying data structures\+:
\begin{DoxyEnumerate}
\item hash\+Map -\/ Hash table based implementation
\item tree\+Map -\/ Red-\/\+Black Tree based implementation
\item JMap -\/ Skip List based implementation
\end{DoxyEnumerate}

Common Features\+:
\begin{DoxyItemize}
\item Key-\/value pair storage with unique keys
\item Full iterator support
\item Customizable comparison/hash functions
\item Customizable allocators
\item Exception safety guarantees
\item Polymorphic usage through map interface
\item Both const and non-\/const element access
\end{DoxyItemize}
