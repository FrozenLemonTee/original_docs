\doxysubsection{Detailed Description}
Implementation of hash\+Map container. 

Provides a hash-\/based map implementation with\+:
\begin{DoxyItemize}
\item Average O(1) time complexity for basic operations
\item Separate chaining collision resolution
\item Customizable hash function and allocator
\item Full iterator support
\end{DoxyItemize}

Key Features\+:
\begin{DoxyItemize}
\item Inherits from map interface for polymorphic use
\item Uses hash\+Table as underlying storage
\item Provides both const and non-\/const element access
\item Supports all standard map operations
\item Exception-\/safe implementation 
\end{DoxyItemize}
