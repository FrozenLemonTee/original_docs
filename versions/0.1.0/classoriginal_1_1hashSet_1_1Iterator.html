Detailed Description

template<typename TYPE, typename HASH = hash<TYPE>, typename ALLOC = allocator<couple<const TYPE, const bool>>>
class original::hashSet< TYPE, HASH, ALLOC >::Iterator

Forward iterator for hashSet.

Provides iteration over hashSet elements while maintaining:

  • Consistent traversal order
  • Safe invalidation detection
  • Const-correct access

Iterator Characteristics:

  • Forward iteration only (throws on reverse operations)
  • Invalidates on rehash
  • Lightweight copy semantics
