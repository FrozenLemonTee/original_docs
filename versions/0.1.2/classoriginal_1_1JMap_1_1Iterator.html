Detailed Description

template<typename K_TYPE, typename V_TYPE, typename Compare = increaseComparator<K_TYPE>, typename ALLOC = allocator<couple<const K_TYPE, V_TYPE>>>
class original::JMap< K_TYPE, V_TYPE, Compare, ALLOC >::Iterator

Forward iterator for JMap.

Provides iteration over JMap elements while maintaining:

  • Sorted traversal order (according to comparator)
  • Safe invalidation detection
  • Const-correct access

Iterator Characteristics:

  • Forward iteration only (throws on reverse operations)
  • Invalidates on container modification
  • Lightweight copy semantics
